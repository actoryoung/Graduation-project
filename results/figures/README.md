1. **图表引用**: 在论文中使用"如图X所示"或"根据图X"等表述
2. **图表位置**: 将图表放在首次引用的段落之后
3. **图表编号**: 按章节编号，如"图3-1"表示第三章图1
4. **图表说明**: 每个图表应有清晰的标题和说明文字
