## ⚡ 性能提示

1. **首次运行**: 需要下载模型，约1-2分钟
2. **后续运行**: 模型已缓存，几秒启动
3. **预测速度**: 文本 <1秒，音频/视频取决于文件大小

## 🛠️ 故障排除

### 问题1: 端口被占用
```bash
# 使用其他端口
streamlit run web/app.py --server.port 8502
```

### 问题2: 模型加载慢
- 正常现象，首次运行需要下载预训练模型
- 后续会使用缓存，速度很快

### 问题3: 浏览器未打开
- 手动访问: http://localhost:8501
- 或: http://localhost:8502 (如果使用8502端口)
